# 🎓 Tuition Finder

## 📌 Project Overview

Tuition Finder is a web application that helps students find nearby tutors based on their location and subject preferences.
The goal is to make it easier for students to connect with the right tutors within a specific range.

---

## 🚀 Current Features

* 📍 Location-based search (within 5km range)
* 🗺️ Map integration
* 🎯 Simple and user-friendly interface

---

## 🛠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB

---

## ⚙️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/Mukund-ai/Tuition-Finder.git
```

2. Navigate to the project folder:

```
cd Tuition-Finder
```

3. Install dependencies:

```
npm install
```

4. Start the server:

```
node script.js
```
